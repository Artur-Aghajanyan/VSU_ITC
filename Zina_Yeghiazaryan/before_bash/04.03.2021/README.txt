Chess
1.This project is about chess. The rules of chess are published by FIDE, chess's international governing body, in its Handbook, which you can find here: 
https://handbook.fide.com/chapter/E012018
2.You can play chess only on Linux kernel OS.
For installation:
1)got in terminal - ctrl+alt+t
2)type: sudo apt install Chess
Then you need to run the program.
